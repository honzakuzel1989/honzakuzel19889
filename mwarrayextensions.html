MWArray.Extensions

Everyone who have ever worked with MWArray class or with inherited classes knows that API of these classes is not very friendly. That was the main reason because I created a friendly, powerfull, fast, extensible and well-tested set of MWArray class extensions.

The set contains only a few self-explaining methods: ToSingleItem, ToArray, ToEnumerableOfArrays, GetField, CreateArray, CreateItem. As you can see below, the methods significantly simplify work with MWArray based classes and improve code writability and readability.

If You want to try these extensions FOR NON COMERTIONAL USE contact me and I will send you dll builded with desired version of MWArray library (if it will be possible - see builds section). Use the same contact if You want to buy MWArray.Extensions library for arbitrary use.

Builds

The MWArray.Extensions library can be build with several versions of MWArray library for several types of operating system (it depends on MATLAB Runtime Version). The prefered version of the MWArray is 2.17.0.0 (MATLAB Runtime codename R2017a version 9.2) and the required operating system is Windows. You must follow these restrictions as same as MATLAB, MATLAB Runtime and MATLAB Compiler restrictions (in readme.txt) otherwise extensions will not work!
